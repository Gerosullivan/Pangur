### Available Skills

Read any skill using the Read tool on the SKILL.md file in the corresponding folder:

- **brainstorming** - Interactive design refinement before coding
- **writing-plans** - Create detailed implementation plans
- **executing-plans** - Execute plan in batches with checkpoints
- **test-driven-development** - RED-GREEN-REFACTOR cycle
- **systematic-debugging** - 4-phase root cause analysis
- **verification-before-completion** - Ensure it's actually fixed
- **subagent-driven-development** - Fast iteration with two-stage review
- **dispatching-parallel-agents** - Concurrent subagent workflows
- **requesting-code-review** - Pre-review checklist
- **receiving-code-review** - Responding to feedback
- **using-git-worktrees** - Parallel development branches
- **finishing-a-development-branch** - Merge/PR decision workflow
- **writing-skills** - Create new skills

To invoke a skill, read its SKILL.md file: `skills/<skill-name>/SKILL.md`
